Ejercicio 1. Elaborar un programa que traduzca de texto a código Morse y viceversa.

Instrucciones.

Validar instalación de JAVA.
0. Validar que se tenga instalada la versión 1.8.0 del compilador de java, así como la versión 1.8.0 del JDK.

Compilación.
1. Navegar hasta el directorio "src/com/puntosingular" del proyecto.
2. Sobre dicha ubicación, ejecutar el siguiente comando: `javac MorseConverter.java`
3. Lo anterior generará la clase MorseConverter.class en el mismo directorio.


Ejecución.
1. Navegar hasta el directorio "src/" del proyecto.
2. Sobre dicha ubicación, ejecutar el siguiente comando: `java com.puntosingular.MorseConverter`
3. Seguir las instrucciones del programa desplegado en pantalla.
